A segurança baseada em perímetro físico ("Castelo e Fosso") tornou-se obsoleta. Em um cenário híbrido, a Identidade é o novo firewall.

4. Resposta Automatizada

A integração entre EDR e NAC reduz o MTTR (Mean Time to Respond). O mecanismo chave é o Radius CoA.

Detecção

EDR/SIEM identifica comportamento anômalo ou malware no endpoint.

Sinalização

API dispara alerta para o NAC Policy Node com ID do host.

Contenção (CoA)

NAC força o switch a mover a porta para VLAN de Quarentena instantaneamente.
